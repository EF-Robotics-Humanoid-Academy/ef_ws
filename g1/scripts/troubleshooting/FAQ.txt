Troubleshooting FAQ
===================

Q: What is `livox_python.py`?
A: A minimal Python wrapper around the Livox SDK C API. It auto-discovers LiDARs on the subnet, connects, starts sampling, and yields each frame as a float32 `(N,3)` array in meters.

Q: I get “Could not locate Livox shared library”. What do I do?
A: Build/install the Livox SDK and ensure `liblivox_sdk.so` (or `LivoxSdk.dll` on Windows) is on `LD_LIBRARY_PATH`/`PATH`.

Q: How do I quickly verify Livox data is streaming?
A: Run `python3 live_points.py`. It displays the raw point cloud in Open3D. Press `Esc` or `Ctrl-C` to quit.

Q: The point cloud looks upside down. How do I fix it?
A: Set `LIVOX_MOUNT=normal` if your sensor is right-side-up. Default is `upside_down`.

Q: What does `mid360_config.json` do?
A: It supplies the SDK2 network configuration for the MID-360, including command, push, point, IMU, and log ports plus the host IP (`192.168.123.222`).

Q: How do I view `utlidar` map state and deskewed cloud?
A: Run `python3 utlidar_map_viewer.py [iface]` and optionally set `--map-state-type` to `heightmap`, `string`, or `lidarstate`. It prints status once per second or uses matplotlib if available.

Q: How do I stream RealSense locally with Python?
A: `python3 stream_realsense.py --width 640 --height 480 --fps 30` (optional `--infra`, `--imu`). It needs `pyrealsense2` and `opencv-python`.

Q: How do I stream RealSense from the Jetson to my laptop?
A: On the Jetson, run `python3 jetson_realsense_stream.py --client-ip <laptop-ip> --width 640 --height 480 --fps 30`. It sends H.264 RGB on UDP port 5600 and H.264 color-mapped depth on 5602.

Q: How do I receive the Jetson stream on the laptop?
A: Run `python3 receive_realsense_gst.py`. It uses GStreamer via PyGObject and OpenCV for display.

Q: My OpenCV build lacks GStreamer support. What then?
A: Use `receive_realsense_gst.py`, which pulls frames via GStreamer appsinks instead of `cv2.VideoCapture`.

Q: How do I visualize saved maps?
A: `python3 visualize_map.py` expects a `maps/` directory containing `.yaml` map files and their referenced `.pgm` images. It prompts you to select a map and renders it with matplotlib.
